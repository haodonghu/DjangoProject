Filephile enables real-time and bidirectional file-sharing. It works on every platform, browser or device, focusing equally on reliability and speed.

Real-time analytics

Superviser & Recommander System based on machine learning.

Binary streaming

It's possible to send any blob back and forth: image, audio, video.

Group organizing

Users can create groups and join other people's groups.

Sharing mechanism

Our sharing mechanism allows everyone to share files to friends.

In the near future, Filephile will be able to convert handwritten notes into text docments with OCR tech. Also, there will be a chat room for each group so that everyone can communicate with others. What's more, machine learning tech will help us with supervising file content so that illegal file will not be sharing on Filephile. More importantly, Fielphile can recommand relative friends and groups whose file is probably in your intereting area.
